# CornerStone — Premium E-Commerce

High-performance e-commerce platform built with Next.js, Supabase, and Turborepo.

## Deployment

This project is optimized for Vercel. Ensure you add the following Environment Variables in the Vercel Dashboard:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `NEXT_PUBLIC_APP_URL`
